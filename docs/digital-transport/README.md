# 数字转盘项目设计文档

本目录是顶级数字转盘（digital transport）项目的设计文档集。

## 设计核心理念

**核心问题**：数字转盘相对 CD 转盘音质差的本质，**不是时钟精度不够、不是数据不对**，而是源端在播放过程中产生的电气活动（电源/地噪、时钟相位调制）和音乐内容相关——这种"音乐相关的污染"通过线缆传到 DAC，调制了 DAC 的模拟参考。

**解决方案**：让源端到 DAC 之间，**只传递"边沿"本身，不传递任何其他东西**。最关键的措施是：

1. **末级孤岛**：把"OCXO + 主时钟重对齐 D-FF + 恒流 CML 驱动 + 变压器隔离"这一小段电路从整机中电气隔离出来，使其无论上游和音乐内容如何变化，对外表现的电气行为完全一致。
2. **主时钟在末级生成、不外传**：消除时钟在传输过程中被污染的可能。
3. **数据预读入 RAM、播放时上游断电**：消除播放期间所有随机 I/O 和 CPU 活动。
4. **galvanic 隔离**：变压器/光纤完全切断地环路。

## 设计推导过程

完整的第一性原理推导记录在 conversation history 中。关键结论：

1. **DAC 只需要两样东西**：样本值（内容必须正确）+ 采样时钟（相位必须精确）
2. **这两样东西需求相反**：数据无时序要求，时钟无内容要求 → 必须分离
3. **时钟应在 DAC 端生成**：时钟离开生成点就只会变差，不可能变好
4. **播放期间应"零传输"**：每一比特跨越物理边界都是污染源 → 数据应预先放到接收端
5. **如果必须传输（实用版）**：只传纯粹的"边沿"，不附带任何协议、元数据、共模电流、共享电源

## 当前阶段

聚焦"**末级输出电路**"——即从主时钟 OCXO 到 BNC/XLR 之间的几厘米电路。这是数字转盘和 CD 转盘音质差距的**根因所在**，也是改造现有数字转盘成本最低、见效最快的切入点。
