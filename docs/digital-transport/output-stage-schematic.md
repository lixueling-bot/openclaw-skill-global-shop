# 数字转盘末级输出电路 — 完整原理图

## 0. 设计目标

让 SPDIF / AES-EBU 输出端在**任何音乐内容下**都表现出完全相同的电气行为：
- 输出方波边沿抖动 < 5 ps RMS
- 共模噪声 < -80 dB
- 电源电流随音乐内容波动 < 0.1%
- 整个末级电路对外只有 3 个电气连接（数据进、电源进、信号出）

## 2. 时钟子电路（U1, Y1, Y2, U2）

### 2.1 双 OCXO + RF 切换

```
                +5V_CLK
                  │
        ┌─────────┴─────────┐
        │                   │
   ┌────┴─────┐       ┌────┴─────┐
   │ Y1: OCXO │       │ Y2: OCXO │
   │ 22.5792M │       │ 24.576M  │
   │ Crystek  │       │ Crystek  │
   │ CCHD-957 │       │ CCHD-957 │
   └────┬─────┘       └────┬─────┘
        │ CLK_44K1         │ CLK_48K
        │                  │
        ▼                  ▼
       RF1                RF2
   ┌──────────────────────────┐
   │   U1: HMC349 RF Switch    │
   │   (or PE42420)            │
   │                           │
   │           CTRL ←── FS_SEL │
   └─────────────┬─────────────┘
                 │ MCLK_RAW
                 │
                 ▼ (送 U2 输入)
```

**Y1, Y2 (OCXO) — Crystek CCHD-957 引脚连接**：

| Pin | Name | Connect |
|-----|------|---------|
| 1   | EN   | +5V_CLK (always enabled) |
| 2   | GND  | GND_CLK |
| 3   | OUT  | → U1 RF1 / RF2 (50Ω trace) |
| 4   | VCC  | +5V_CLK (with π-filter: 10Ω + 100nF + 1nF) |

每颗 OCXO 的 VCC 脚单独串一颗 10Ω + 三颗 100nF/1nF/100pF 多值并联去耦。

**U1 (HMC349 SP2T RF Switch)**：

| Pin | Name | Connect |
|-----|------|---------|
| 1   | RFC  | → U2 input (50Ω microstrip) |
| 2   | RF1  | ← Y1 OUT (50Ω microstrip) |
| 3   | RF2  | ← Y2 OUT (50Ω microstrip) |
| 4   | CTRL | ← FS_SEL (3.3V logic) |
| 5   | VDD  | +5V_CLK |
| 6,7 | GND  | GND_CLK |

### 2.2 时钟 Buffer / Fanout — U2 (ADCLK948)

| Pin | Name      | Connect |
|-----|-----------|---------|
| CLK | input     | ← U1 RFC (LVCMOS 输入模式) |
| OUT0+ / OUT0- | LVPECL | → U3 CK / CKbar (重对齐 DFF 时钟) |
| OUT1+ / OUT1- | LVPECL | → 经 ADuM320N 隔离 → 上游 FPGA MCLK |
| OUT2-5 | unused | 100Ω 终端到 GND |
| VS  | +5V_DIG   | |
| GND | GND_DIG   | |

**OUT0 走差分对，长度匹配 ±0.5mm，100Ω 差分阻抗。**

## 3. 重对齐 D 触发器 — U3 (NB7L72M)

这是整个设计的**精神中枢**。所有上游污染在这里被主时钟"洗一遍"。

```
        ┌─────────────────────────┐
        │      U3: NB7L72M         │
        │                          │
   D ──→│ Pin 7                    │
  Db ──→│ Pin 6                    │
        │                          │
  CK ──→│ Pin 9                    │
  CKb ─→│ Pin 8                    │
        │                          │
        │                  Pin 3 │──→ Q
        │                  Pin 4 │──→ Qb
        │                          │
        │  VCC: Pin 2,10  +5V_DIG  │
        │  VEE: Pin 1,5   GND_DIG  │
        └──────────────────────────┘
```

**关键时序要求**：
- 数据 D 必须在 CK 上升沿**之前 ≥ 200 ps** 稳定（NB7L72M tsetup spec = 100ps，留 2× 裕量）
- D 信号从 U5 输出到 U3 输入的走线**等长于** CK 信号从 U2 输出到 U3 输入的走线 ± 5mm
- D 信号路径上**绝不放任何电容、磁珠、电阻**（除了 100Ω 差分终端）

**去耦**：VCC 脚下放 100nF X7R + 10nF C0G + 1nF C0G + 100pF C0G，全部 0402 封装，紧贴芯片脚（< 1mm）。

**Q / Qbar 后**串 50Ω 阻尼电阻（每条单线），然后差分送 U4。

## 4. CML 恒流输出驱动 — U4

### 4.1 方案 A：集成芯片 SY58025U

```
        ┌──────────────────────────┐
        │      U4: SY58025U         │
        │                           │
  Q  ──→│ INP                       │
  Qb ──→│ INN                       │
        │                           │
        │                  OUTP ───→│ CML+  ──→ T1/T2 初级
        │                  OUTN ───→│ CML-  ──→ T1/T2 初级
        │                           │
        │  VCC: +5V_DRV             │
        │  VEE: GND_DRV             │
        └───────────────────────────┘
```

OUTP / OUTN 各串 50Ω（如果变压器初级是 100Ω 差分），变压器初级中点经 200Ω 接 +5V_DRV/2 偏置点。

### 4.2 方案 B：离散 CML（极致路线，推荐）

```
                  +5V_DRV
                     │
              ┌──────┴──────┐
              │             │
            [R1]          [R2]      R1=R2=50Ω 1% (Susumu RG)
              │             │
              ├─── CML+ ────┤ → T1/T2 初级
              ├─── CML- ────┤
              │             │
            [Q1]          [Q2]    BFR93AW (SOT-323, fT=5GHz)
              │             │
   Q   ──┬───┘            └─┬── Qb     (经 200Ω 串联送基极)
   Qb  ─┤                    │ Q
        │                    │
        └──────┬──────────────┘
               │
              [Q3]    BF862 JFET (恒流源)
               │ ID ≈ 8mA (R_S 调整)
              [Rs]   100Ω
               │
              GND_DRV
```

**元件清单（单端口，差分对）**：

| 位号 | 型号 | 用途 |
|------|------|------|
| Q1, Q2 | BFR93AW (BFG403W 更佳) | 差分对，高 fT NPN |
| Q3 | BF862 (JFET) 或 LM334 | 恒流源 |
| R1, R2 | 50Ω 0.1% 薄膜 | 集电极负载 |
| Rs | 100Ω 0.1% | 恒流源调节 |
| Rb1, Rb2 | 200Ω | 基极串入 |

**电源去耦（紧贴 R1/R2 顶端）**：100μF 钽 + 10μF X7R + 100nF + 1nF + 100pF

## 5. 输出变压器与连接器（T1, T2）

### 5.1 SPDIF 路径（T1: SC916）

```
   CML+ ──┬──┐
          │  )||(
          │  )||(   T1: Scientific Conversion SC916
          │  )||(   1:1, 75Ω
   CML- ──┴──┘
              │
              ├──→ HOT
              │
            [R3] 75Ω 0.1%
              │
              ├──→ GND_OUT (chassis)
              │
            [J1] BNC (Amphenol B6F30N3)
```

变压器次级到 BNC 的走线**严格 75Ω 单端微带**，长度 < 30mm，正下方完整地平面。

### 5.2 AES 路径（T2: LL1572）

```
   CML+ ──┬──┐
          │  )||(
          │  )||(   T2: Lundahl LL1572
          │  )||(   1:1, 110Ω 平衡
   CML- ──┴──┘
              ├─── XLR Pin 2 (HOT)
              │

              ├─── XLR Pin 3 (COLD)
              │
```

## 6. 电源域（独立线性电源）

### 6.1 电源树

```
   220V AC
     │
     ▼
  ┌─────────────────────────────────────┐
  │ 电源变压器副绕组 #N (8VA, 8V AC)    │  ← 专用绕组，独立屏蔽
  └────────────────┬────────────────────┘
                   │
                   ▼
              [D1-D4]  MBR1045 ×4 (软恢复肖特基)
                   │
                   ▼
              ─┴─ C1 4700μF/25V (Nichicon KZ 或 Mundorf)
                   │
                   ▼
              [L1]   共模扼流 100μH
                   │
                   ▼
              ─┴─ C2 4700μF/25V
                   │
                   ▼ +VRAW ≈ +11V
              [U6: LM317]   预稳到 +8V
                   │
                   ▼ +8V_PRE
                   │
        ┌──────────┼──────────┬──────────┐
        │          │          │          │
        ▼          ▼          ▼          ▼
   [LDO1]     [LDO2]     [LDO3]     [LDO4]
   LT3045     LT3045     LT3045     LT3045
   +5V_CLK    +5V_DIG    +5V_DRV    +3.3V_ISO
   (Y1,Y2)    (U2,U3)    (U4)       (U5)
```

### 6.2 LT3045 标准电路（每路独立）

```
            +8V_PRE
              │
              ▼
       Pin 1: IN
            ┌────────────┐
            │            │
   PROG ──[Rset]──┐      │
   (Pin 4)        │     LT3045
                  │            │
            Pin 7: OUT ────┬──→ +5V_xxx
                  │        │
                  │      C_out
                  │      22μF + 100nF + 1nF
                  │        │
                  │       GND
            Pin 5: SET
                  │
                  C_ref 10nF (低漏)
                  │
                 GND
```

**Rset 选值**（设定输出电压）：
- +5V output: Rset = 50kΩ (Vout = 100μA × Rset)
- +3.3V output: Rset = 33kΩ

**关键细节**：
- C_ref（Pin 5 到 GND）用 **C0G 10nF**，绝不用电解或低质陶瓷（漏电会推高噪声）
- C_out 用 22μF X7R + 100nF C0G + 1nF C0G 多值并联
- LT3045 EP 散热焊盘直接焊到大铜区（≥ 4cm²），保证 dropout 余量

### 6.3 地的处理（关键）

```
    GND_CLK ──┐
              │
    GND_DIG ──┼──→ Star Point ──[10Ω 或 0Ω 跳线]──→ Chassis GND
              │   (在 LDO 输出
    GND_DRV ──┤    回流处)
              │
    GND_ISO ──┘ (隔离器次级地，仅经 Si8645 内部连接，物理不连)
```

- 末级孤岛的四个地（CLK/DIG/DRV/ISO 中的输入侧）在 LDO 输出回流点**单点星接**
- 这一点**用一根 10Ω 电阻或 0Ω 0805 跳线**接到机壳地
- 隔离器 U5 的**次级地**（ISO 端）和末级孤岛地物理隔开，仅经 Si8645 内部电容耦合
- **绝对不要**让数字部分的地通过 Si8645 直接连过来——这是孤岛的关键防线

## 7. 上游数据隔离 — U5 (Si8645BB)

```
   ── 上游域（脏） ──────┃────── 末级孤岛（净） ───────
                          ┃
   SDATA_IN ──→ A1   B1  ──→ D  (送 U3 Pin 7)
   BCLK_IN  ──→ A2   B2  ──→ (可选, 调试用)
   LRCK_IN  ──→ A3   B3  ──→ (可选, 调试用)
   (反向)   ←── A4   B4 ──── MCLK_OUT (送上游)
                          ┃
   +3.3V_UPS ──→ VDDA  VDDB ←── +3.3V_ISO
   GND_UPS  ──→ GNDA  GNDB ←── GND_ISO
                          ┃
                Si8645BB
```

**Si8645 输出本身有 < 1ns 抖动**——但没关系，下游 U3 (NB7L72M) 会用主时钟把它"洗干净"。

**Si8645 的 B 侧（末级侧）电源是 +3.3V_ISO，由 LDO4 单独供给，地为 GND_ISO**，物理上和 GND_DIG/CLK/DRV 通过 0Ω 跳线在 LDO4 输出点单点连接。

## 9. PCB 叠层与布局

### 9.1 推荐叠层（6 层）

| 层 | 用途 | 厚度 |
|----|------|------|
| L1 (Top) | 信号 / 元件 | 35μm Cu |
| L2 | **完整 GND** | 17μm Cu |
| L3 | 信号 (低速 / 控制) | 17μm Cu |
| L4 | 电源平面 (+5V_CLK, +5V_DIG, +5V_DRV) | 17μm Cu |
| L5 | **完整 GND** | 17μm Cu |
| L6 (Bot) | 信号 / 元件 | 35μm Cu |

- 板厚 1.6mm，TG170 FR4 或 Rogers RO4350B（高端）
- L1-L2 距离 0.2mm（用于精确控制 50Ω/100Ω 阻抗）
- L4 电源平面**按区域分割**：左 1/3 给 CLK 域、中 1/3 给 DIG 域、右 1/3 给 DRV 域，**域间留 2mm 空隙**

### 9.2 布局分区

```
  ┌────────────────────────────────────────────────────────────┐
  │   PCB 顶视图（建议尺寸 100mm × 80mm）                       │
  │                                                              │
  │  ┌─────────────┐  ┌──────────────┐  ┌──────────────────┐  │
  │  │  电源区      │  │  时钟域       │  │  数据/驱动域     │  │
  │  │              │  │              │  │                  │  │
  │  │  环牛副绕组  │  │  Y1 Y2  U1   │  │   U5            │  │
  │  │  整流滤波    │  │  U2          │  │   U3            │  │
  │  │  LDO1-4      │  │              │  │   U4            │  │
  │  │              │  │              │  │   T1 T2         │  │
  │  └─────────────┘  └──────────────┘  └──────────────────┘  │
  │                                                              │
  │   ←──── 25mm ───→ ←──── 30mm ───→  ←──── 35mm ───→       │
  │                                                              │
  │                                              [BNC]  [XLR]  │
  └────────────────────────────────────────────────────────────┘
```

### 9.3 关键布局规则

1. **OCXO 物理位置**：放在板边缘、远离电源整流、下方完整地平面、不被走线穿过
2. **OCXO → U2 → U3.CK 走线**：尽量直线，总长度 < 25mm
3. **U3 → U4 → T1/T2 走线**：直线，总长度 < 20mm，全程不变层
4. **变压器 T1, T2**：距离 BNC/XLR 连接器 < 30mm
5. **LDO 输出去耦电容**：紧贴芯片 VCC 脚 < 1mm
6. **关键差分对**：保持完整参考平面（不跨过分割平面），如必须跨过用 0.1μF 跨接电容跨平面回流
7. **CML 输出走线下方禁止开槽**

### 9.4 关键差分对阻抗（典型 6 层 FR4，L1-L2 间距 0.2mm）

| 差分对 | 线宽 W | 线距 S | 阻抗 |
|--------|--------|--------|------|
| 100Ω LVPECL (CK, D, Q) | 0.18mm | 0.18mm | 100Ω ±10% |
| 100Ω CML (CML+/-) | 0.18mm | 0.18mm | 100Ω ±10% |
| 110Ω AES (XLR) | 0.17mm | 0.22mm | 110Ω ±10% |
| 75Ω SE (BNC) | 0.30mm (over L2 GND) | — | 75Ω ±10% |

具体值需用 Saturn PCB Toolkit / Polar Si9000 按实际叠层精算。

## 10. 测试点和调试接口

| 测试点 | 信号 | 期望波形 |

## 11. 已知风险与对策

| 风险 | 影响 | 对策 |
|------|------|------|
| OCXO 上电温度未稳定 | 频率漂移、相位噪声升高 | 加 ready 指示，5 分钟预热再播放 |
| LT3045 启动顺序 | LDO 输出冲击 | 加 100kΩ pull-down，确保关机放电 |
| 上游和孤岛地参考差异过大 | Si8645 共模超限 | 加 ESD TVS + 共模扼流圈在隔离器前 |
| 变压器寄生电容导致高频馈通 | EMI 上升 | T1/T2 选屏蔽款，外壳接机壳地 |
| 振动调制 OCXO | 微相位调制 | OCXO 加硅胶减震套，PCB 加四个 Sorbothane 角脚 |

## 12. 验证计划

**第一阶段（板级验证）**：
1. 上电不带负载，测各 LDO 输出噪声（频谱仪 9kHz-1GHz）
2. 注入测试 PRBS 数据，测 BNC/XLR 输出眼图
3. 测输出边沿抖动（用 Tektronix DSA8300 或 Keysight DCA-X）
4. 目标：jitter < 5 ps RMS（10kHz-100kHz 积分范围）

**第二阶段（系统验证）**：
1. 接 Mola Mola Tambaqui 或 dCS Bartók 等高级 DAC
2. 用 J-Test 信号（AP-2722）测 DAC 端解析出的 jitter
3. 主观对比：vs Esoteric P-02X CD 转盘走相同 DAC

**目标**：主观听感差距 < 5%（盲听 ABX 不能可靠区分）
