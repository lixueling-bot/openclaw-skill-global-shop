# Global Shop Skill 🛍️

全球购物比价与下单技能，让 AI 帮你在全球电商平台搜索、比价、下单。

## 功能特性

- 🔍 **全球搜索** — 并发搜索 Amazon、eBay、AliExpress、天猫国际、京东全球购
- 📊 **智能筛选** — 自动聚合结果，展示 3~5 个最优选项
- 💳 **统一支付** — 支持信用卡统一支付，可记住支付方式
- 🔐 **平台账户** — 支持预存平台账户，快速登录购买
- 🛡️ **安全可靠** — 不存储 CVV，所有支付需用户确认

## 安装方式

### 方式一：通过 .skill 文件安装

下载 `global-shop.skill` 文件，双击或拖入 OpenClaw 即可安装。

### 方式二：通过 SkillHub 安装

```
skillhub install global-shop
```

## 使用方式

| 用户说 | AI 执行 |
|--------|---------|
| "我想买 AirPods" | 搜索全球平台，展示最优选项 |
| "最便宜的 iPhone 在哪" | 直接显示最低价 |
| "用我的 Amazon 账户买" | 调用预存账户 |
| "查看我的信用卡" | 显示已存支付方式 |
| "删除已存信用卡" | 清除支付信息 |

## 工作流程

```
用户表达购买意图
    ↓
AI 搜索全球平台（Amazon/eBay/AliExpress/天猫/京东）
    ↓
展示 3~5 个最优选项（价格/配送/评分对比）
    ↓
用户选择 → AI 打开对应平台
    ↓
用户确认支付 → 完成下单
```

## 安全原则

- ✅ 用户主动确认后才执行支付
- ✅ 信用卡仅存储末4位，不存储 CVV
- ✅ 不存储任何平台密码
- ✅ 所有敏感信息本地加密存储
